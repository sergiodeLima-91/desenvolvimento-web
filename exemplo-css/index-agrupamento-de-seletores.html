Agrupamento de Seletores

Podemos aplicar a mesma regra para vários seletores diferentes. Para isso, podemos separar os seletores utilizando virgula(,).

Assim, precisamos ficar copiando e colando os mesmos códigos para todos, mas sim fazer blocos de CSS, reutilizando as mesmas regras!
