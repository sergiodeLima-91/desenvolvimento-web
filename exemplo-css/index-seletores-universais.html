Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID:

  • Seletor por Tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor de Classe: Busca elementos através do atributo "class"
  • Seletor Universal:Seleciona todos os elementos HTML
  • Seletor de Atributo: Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos buscar atributos com um valor específico
